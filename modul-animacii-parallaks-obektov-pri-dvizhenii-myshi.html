Модуль анимации (параллакс) объектов при движении мыши

Категории: Документация «Чертоги Фрилансера» v3.0.0

Шикарный модуль для создания интерактивной анимации на вашей странице. Различные объекты могут плавно передвигаться реагируя на перемещения курсора мыши.

Подключение функционала

[HTML] К объекту, который следует анимировать, необходимо добавить атрибут data-prlx-mouse

<img src="img/cover.jpg" data-prlx-mouse alt="Картинка">

[JS] В файле js/app.js раскомментировать строку import ‘./libs/parallax-mouse.js’

Дополнительные настройки

  1. data-prlx-cx=коэффициент_х (значение по умолчанию 100) — управление процентом сдвига по оси X
  2. data-prlx-cy=коэффициент_y (значение по умолчанию 100) — управление процентом сдвига по оси Y
  3. data-prlx-a=скорость анимации (значение по умолчанию 50) — управление скоростью анимации
  4. data-prlx-dxr — движение в противоположном направлении по оси X
  5. data-prlx-dxy — движение в противоположном направлении по оси Y

Расположение и дополнительные данные

Функционал находится в js/libs/parallax-mouse.js.
