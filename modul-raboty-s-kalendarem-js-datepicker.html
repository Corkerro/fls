Модуль «Календарь» (js-datepicker)

Категории: Документация «Чертоги Фрилансера» v3.0.0

Функционал позволяет выводить календарь для поля ввода (input).

Подключение функционала

[JS] В файле js/app.js раскомментировать строку import ‘./files/forms/datepicker.js’
[SCSS] Раскомментировать строку @import «datepicker»; в файле src/scss/base/forms/forms.scss — это подключит базовые стили календаря. Отредактировать стили под свои нужды можно в файле src/scss/base/forms/datepicker.scss

Использование модуля

[HTML] К тегу <input> добавляем HTML-атрибут data-datepicker. 

<input data-datepicker type="text" name="">

Настройки и функционал модуля

Настройки модуля можно изменять в файле src/js/files/forms/datepicker.js. Документация плагина https://www.npmjs.com/package/js-datepicker
